🏗️ Usta-Market
Usta-Market is an open-source construction materials marketplace built for the Uzbekistan market. It connects buyers, sellers, and contractors in one platform — making it easier to find, compare, and purchase building materials across Uzbekistan.
🌍 Why This Matters
Uzbekistan's construction sector is one of the fastest growing in Central Asia, yet there is no dedicated digital marketplace for construction materials. Usta-Market fills this gap by providing:

A transparent marketplace for construction materials
Real-time pricing and availability from local suppliers
A platform accessible to small contractors and individual buyers

✨ Features

🛒 Product listings with categories (cement, steel, tiles, wood, etc.)
🔍 Search and filter by price, location, and supplier
💬 Real-time chat between buyers and sellers (Socket.io)
📦 Order management system
🔐 Authentication & user roles (buyer / seller / admin)
📊 Seller dashboard with analytics

Installation
bash# Clone the repository
git clone https://github.com/AhmadOdilov/usta-market.git
cd usta-market

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run development server
npm run dev
